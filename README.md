# ActionBoard (Dynamic ActionBar & Display Engine) 📊

**ActionBoard** is a lightweight, real-time actionbar display engine built for **Paper 1.21.x** servers.

---

## Features

- Real-Time Actionbar: Displays online players, max player capacity, and player ping.
- Adventure MiniMessage: Modern rich-text color gradient formatting.
- Ultra Lightweight: Low CPU overhead.
